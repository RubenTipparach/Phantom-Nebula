================================================================================
BUILD WORKFLOWS ENABLED
================================================================================

Desktop Native:
$ dotnet build PhantomNebula -c Release
→ Output: PhantomNebula/bin/Release/net8.0/PhantomNebula

WebAssembly (Browser):
$ ./build-wasm.sh && ./bin/wasm/serve.sh
→ Output: PhantomNebula/bin/Release/net8.0/browser-wasm/AppBundle/
→ Browser: http://localhost:8000

Interactive:
$ ./build-web.sh
→ Menu with 6 options

Platform-Specific:
$ dotnet build PhantomNebula -r [win-x64|osx-x64|osx-arm64|linux-x64]

Self-Contained:
$ dotnet publish PhantomNebula --self-contained

Single File:
$ dotnet publish PhantomNebula -p:PublishSingleFile=true

✓ Better performance

✓ Single unified codebase for all platforms
✓ Faster development and maintenance
✓ Easier bug fixes across platforms
